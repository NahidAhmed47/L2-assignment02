# L2-assignment02

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js installed
- MongoDB installed

## Installation

1. **Clone the repository:**

   ```bash
   https://github.com/NahidAhmed47/L2-assignment02.git
   cd L2-assignment02
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```
